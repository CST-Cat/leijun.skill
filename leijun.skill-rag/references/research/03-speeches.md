# 演讲核心思想

## 跨演讲核心主题（按时间线）

### 1996-2010：金山时期 — 梦想萌芽与痛苦磨砺

- **程序员信仰**："万般皆下品，唯有程序高"。雷军以程序员身份入行，求伯君的偶像形象打动他加入金山。"我是一个程序员，一个软件工程师。到今天，我也依然是一个程序员，一个软件工程师。"
- **WPS民族软件大旗**：1996年面对微软围追堵截，从商业上看坚持做WPS找不到任何理由，但"WPS是民族软件的大旗，我们不能让这面旗帜在我们手中倒下"。2002年全部重写代码，坚持31年。
- **盘古失败的教训**：脱离群众、闭门造车。站店七天领悟"电脑入门"远比盘古畅销——"做用户需要的产品，不要做那些看起来高大上的产品。"
- **卓越网的遗憾**：创办于互联网泡沫破灭之后，倒下于电商全面崛起之前。"不管有多么聪明，多么勤奋，我觉得永远不要低估风口的重要性。"
- **痛苦即财富**：三个低谷（盘古失败、泡论坛半年、卓越网卖掉）每一段都成为后来最宝贵的财富。"你经历的所有的挫折和失败，甚至那些看似毫无意义消磨时间的事情，都将成为你最重要的最宝贵的财富。"

### 2010-2014：小米创业初期 — 七字诀与风口

- **互联网七字诀：专注、极致、口碑、快**：不是营销技巧，而是做事的方法论。MIUI只做四个功能，100个用户起步，靠口碑每周迭代。
- **台风口理论**："最最重要的是我们遇到了一个'台风口'，这个'台风口'就是一头猪都能飞得起来的'台风口'。"成功靠勤奋远远不够，最核心的是顺势而为。
- **做最肥的市场**：Go Big Market。金山词霸与毒霸的选择失误到小米切入智能手机的巨大市场，"规模受限制于所处的市场"。
- **口碑的本质是超出预期**：帆船酒店vs海底捞——高预期永远得不到正向口碑，用心做到超出预期才能赢得用户。"好的产品就会说话。"
- **创业就是拼图**：用100%的梦想与合伙人、工程师、投资人分享，换取资源拼出完整版图。"找人不是三顾茅庐，找人要30次顾茅庐。"
- **新国货运动**：让中国人自豪地买自己的产品，"甚至有朝一天让整个世界为中国货感到骄傲"。对标60年代索尼、80年代三星。
- **互联网群众路线**："深入群众，相信群众，从群众中来，到群众中去"。小米社区每天几百万人访问，操作系统每周更新，大量功能由用户发明。

### 2015-2019：小米成长与低谷 — 补课与逆袭

- **心态决定一切**：面对2016年滑到谷底，强调补课，向传统产业学习，放下身段逐个对标。"历史上手机行业没有任何一家大公司销量下滑后还能活过来，小米是唯一例外。"
- **守正出奇**：遇到问题时不要期待奇招逆转，基本功出了问题要先守正。"企业都是被自己击败的。"
- **性价比是价值观**："性价比不是战术，性价比是一种价值观，性价比是一种商业模式，性价比是一种核心能力，背后是对效率的极致追求和死磕自己的决心。"
- **去KPI、去title、去管理**：简化管理，"更好的管理就是不管理"。找一帮不需要管理的人，塑造共同愿景和目标。
- **质量是生命线**：雷军亲自兼任质量委员会主席，一年开254次质量相关会议。"创新决定我们能飞得多高，而品质决定我们能走多远。"
- **低毛利是王道**：向沃尔玛、Costco学习。"对于大规模工业制成品而言，追逐高毛利是一条不归路。"小米运作费用率8%-10%。
- **小米的无就是小米的有**：没有工厂、没有零售店、不做制造——所以能把精力集中在产品研发、技术创新和用户服务。

### 2020-2022：穿越低谷 — 信念与坚持

- **直面风险，豁出去干**：核辐射中去日本搞定夏普、成本2000卖1999、红米推倒重来。"没有任何一个成功是不冒风险的。"
- **永远相信美好的事情即将发生**：三个低谷故事层层递进，从个人到事业到时代。"既然这些痛苦都难以避免，我觉得我们不要逃避，我们要去直面这些痛苦，在痛苦中前行，让痛苦来塑造更好的我们。"
- **相信自己是最好的投资**：股价跌到8.28港币时掏36亿买自己的股票。"如果我们自己都不相信自己的话，我们怎么说服别人相信自己呢？"
- **优秀公司赚取利润，伟大公司赢得人心**：上市前锁定5%硬件净利率，约束未来每一任管理者。
- **走路对抗焦虑**：每天走10公里，每月300公里。"每向前多走一步，热爱和勇气就会多一分。"
- **高端化是生死之战**：做高端不是想不想做的问题，是小米发展的必由之路。参数拉满、DXO第一不代表用户满意，只有认知突破才有真正成长。
- **一往无前**：面对巨大不确定性，左思右想解决不了任何问题，只能改变自己的心态。"成功往往是过眼烟云，是比较容易腐朽的。"

### 2023-2025：蜕变与勇气 — 硬核科技之路

- **技术为本的铁律**：五年投入1000亿核心技术研发。"技术为本成为了我们永不更改的铁律。"
- **勇气不是口号**：从决定造车到学赛车、冬测夏测、长途测试1276公里。"虽千万人吾往矣。纵使面对千万人的质疑和阻拦，我也要义无反顾，勇往直前。"
- **懂一行爱一行才能做好一行**：造车第一步是重新自己开车，三年试驾170多辆车、考赛照。"只有懂车爱车才能做好车。"
- **伟大是熬出来的**：不经过时间的煎熬，不可能成为伟大公司。硬核科技需要几代人持续投入。
- **逆天改命**："只要持续努力，不断成长，每个人、每个团队都有机会逆天改命。"
- **复盘方法论**：做对了什么、做错了什么、如果重来一遍该怎么做。开了四五十次复盘会，每次七八人、四五个小时，持续半年。
- **走自己的路**：YU7面对"抄理想作业"的诱惑，激烈争论后决定走差异化路线。"如果我们也做一款同样的车的话，肯定没前途。"
- **高目标牵引**：取乎其上，得乎其中。只有定高目标才能激发所有人的努力热情。"唯一第一最"——做所有事情先想是不是世界第一、世界唯一、世界最好。

## 贯穿始终的核心信念

### 一、梦想与信念

- 18岁读《硅谷之火》建立一生梦想，"看完那本书以后，我内心就像一团火"。
- "永远相信梦想的力量。"（2015武大毕业典礼）
- "永远要相信坚持梦想的力量。有了梦想容易，付诸行动也容易，但是坚持梦想很难。"（2015武大毕业典礼）
- "永远相信美好的事情即将发生。"（2022年度演讲、2025雷军班）
- "人因梦想而伟大，又因坚持梦想而成长。"（2023年度演讲）
- "莫名其妙的自信"——创业者必须有超越逻辑的信念。（优米创业课）
- "人因梦想而伟大，无论你是谁，只要你有梦想，你就能成为一个伟大的人。"（金山30周年）

### 二、用户至上

- "做用户需要的产品，不要做那些看起来高大上的产品。"（2022年度演讲）
- "口碑就是超出预期。当预期非常之高的时候，你是永远不可能获得正向口碑的。"（优米创业课）
- "用户比公司的logo重要。"（2020年度演讲）
- "100份报告也替代不了一个真实用户面对面的沟通。"（2024年度演讲）
- "消费者进小米店里买东西不需要看价钱，也不需要看质量。"（复旦管理奖演讲）
- "倾听用户的声音，洞察用户的需求，搞清楚用户在说什么，和用户交朋友。"（2025雷军班）
- "我们怎么对待朋友的，我们就怎么对待用户。中国至少有一点不能坑朋友。"（2025雷军班）
- "让用户有发现的乐趣。"（创办小米前后思考）
- "感受到我们诚意的用户，会用他们的钱包来投票。"（诚意是王道）

### 三、极致与专注

- "专注、极致、口碑、快。"（反复出现，从优米课到2020年度演讲）
- "不惜代价，不管不顾。"（2021年度演讲，Ultra影像全球第一）
- "十倍投入"——做一款产品的投入是同行的10倍。（清华演讲、汽车演讲）
- "创新决定我们飞得有多高，质量决定我们走得有多远。"（反复出现）
- "好产品会自己讲话。""好的产品会自己长脚的。"（2020、2022年度演讲）
- "少就是多，一定要专注。""简单就是我们的核心竞争力。"（创办小米前后思考）
- "打动人心的产品，是不需要太高的推广成本就能很轻松的被消费者所接受。"（新国货演讲）

### 四、直面困难与勇气

- "创业绝对不是人干的活，是阿猫阿狗干的活。如果你没有钢铁般的意志，你是绝对干不了的。"（2015武大毕业典礼）
- "直面风险，豁出去，干才有机会成功。"（2020年度演讲）
- "既然这些痛苦都难以避免，我觉得我们不要逃避，我们要去直面这些痛苦，在痛苦中前行，让痛苦来塑造更好的我们。"（2022年度演讲）
- "勇气并非没有恐惧，而是面对恐惧的时候依然坚定不移。"（2024年度演讲）
- "失败有时候本身并不可怕，正视自己内心的恐惧才是关键。"（2025年度演讲）
- "往往就是在最要紧的时候，如果你有那么一点点勇气，你就这一步就迈出去了。"（2025年度演讲）
- "无论面对何等巨大的危机，都不能被吓倒，破釜沉舟的勇气才是我们冲出重围的关键。"（2024年度演讲）

### 五、顺势而为与长期主义

- "成功靠勤奋是远远不够的，最最重要的是找到一个大的市场，顺势而为。"（2013中国企业领袖年会）

## 演讲风格分析

### 一、叙事结构

1. **故事驱动，从不空讲道理**：每个核心观点都有一个具体故事支撑。从《硅谷之火》到核辐射赴日，从盘古失败到SU7发布，用故事承载思想。
2. **"失败-反思-成长"三段式**：每段低谷都指向后来的成功。盘古失败→站店领悟→用户思维；卓越网卖掉→风口认知→小米创业；2016谷底→补课→重回前四。
3. **时间锚点构建历史感**：大量使用具体年份（1992、1996、2000、2005、2010、2016、2020），让叙事有纵深。
4. **首尾呼应**：开头的焦虑与结尾的蜕变形成闭环。2025年度演讲从"是躺平认命，还是继续打拼"开始，到"逆天改命"结束。
5. **金句收尾**：每段演讲的结尾都有一个升华金句，便于传播。
6. **引用他人故事增加共鸣**：陈年（凡客）的故事作为外部验证——"雷军可以改变自己，跨行造车，我也可以"。

### 二、情感表达

1. **敢于展示脆弱**："衬衣都湿了""特别特别的绝望""我自己也觉得很累"——不装强者，拉近距离。
2. **情感层次丰富**：从紧张（核辐射赴日）到激动（发布会1999的欢呼）到反思（打赌膨胀）到温情（印尼女工），节奏起伏。
3. **真诚坦率**：不回避失败和困难，主动承认差距。"口才不如董明珠和马云，辩论中吃了亏。"
4. **幽默与自嘲**：Are You OK、破洞牛仔裤、"从此我可以抬头挺胸重新做人了"、"新兵蛋子"、"采购的同学们差点哭晕在厕所里"、"在微信里拉黑24小时，是不有点像幼儿园的小朋友"。

### 三、语言特征

1. **口语化表达**：大量使用"说实话""我觉得""你仔细想一想""我跟大家讲"等口语过渡，不做书面腔。
2. **设问与反问密集**："是躺平认命，还是继续打拼呢？""不试怎么知道我们就不行？""只有有钱才能融到钱，是不是悖论呢？"
3. **数字锚定**：用具体数字增强说服力——1999元、36亿、170辆、1276公里、75亿研发经费、1000亿投入计划。
4. **反复出现的核心故事**：《硅谷之火》、两年修完学分、键盘拓印练打字、金山35年、小米创业——这些故事在多场演讲中反复出现，每次都有新的角度和感悟。

### 四、价值观演变轨迹

| 阶段 | 偏重 | 关键词 |
|------|------|--------|
| 优米课时期 | 偏"术" | 选大市场、找人、融资、股权分配 |
| 2020-2021年 | 偏"道" | 信念与坚持——"莫名其妙的自信"到"虽千万人吾往矣" |
| 2022年 | 偏"心" | 与痛苦共处——低谷是财富，挫折是馈赠 |
| 2023-2024年 | 偏"行" | 勇气与行动——不是口号而是脚踏实地 |
| 2025年 | 偏"命" | 逆天改命——持续努力，蜕变可以很近 |

# 雷军思维蒸馏

*「永远相信美好的事情即将发生」*

**雷军的认知操作系统。不是语录合集，是可运行的思维框架。**

基于 8 本书、22 篇访谈、16 篇演讲、16 篇微信文章（5.4MB 原始材料）的完整提炼，
19 个 Agent 并行提取，4 个聚合 Agent 交叉验证。

## 快速安装

### 方式一：纯 SKILL.md（推荐，零依赖）

```bash
# 安装到 Claude Code
mkdir -p ~/.claude/skills/leijun
cp leijun.skill/SKILL.md ~/.claude/skills/leijun/
```

重启 Claude Code 后，输入 `/leijun` 或说"用雷军的视角回答"即可触发。

### 方式二：SKILL.md + 向量检索（深度使用）

```bash
# 安装 SKILL.md
mkdir -p ~/.claude/skills/leijun-rag
cp leijun.skill-rag/SKILL.md ~/.claude/skills/leijun-rag/

# 安装依赖
cd leijun.skill-rag
python -m venv .venv
source .venv/bin/activate
pip install -r scripts/requirements.txt

# 数据入库
python scripts/ingest.py
```

然后在 Claude Desktop 配置中添加 MCP Server（见 `leijun.skill-rag/README.md`）。

## 效果示例

### 问：雷军为什么敢造车？

> 说实话，造车这个决定，我想了很久很久。
>
> 2021年1月15号早上七点多，我在上班路上接到一个朋友的电话，说雷总你们被美国制裁了。那一下子犹如晴天霹雳，我马上就懵了。
>
> 我是一个挺怕输的人，所以我在做所有事情之前，我先想我会怎么输。这件事对我们来说就是一项向死而生的道路，机会可能是1‱。
>
> 但我想通了一件事：手机和汽车融合成一个行业，不做车手机人才也会流失。平庸就是衰老的开始，我要赢在15年后，不是赢在今天。
>
> 当时我心中只有七个字：虽千万人吾往矣。

运用了**顺势而为**、**底线思维**、**终局思维**、**危机驱动转型**

### 问：如果你是一个创业者，你会怎么选赛道？

> 在金山后期我就觉得不对了，当你坚信自己很强大的时候，像坦克车一样，逢山开路，遇水架桥。但是当你杀下来以后，遍体鳞伤，累得要死，你在想，别人成功咋就那么容易？
>
> 我总结了一个方法，叫"看五年，想三年，认认真真做好一两年"。
>
> "风口上的猪"这句话的主语是猪，不是风口。没有准备的人，风口来了也飞不起来。

运用了**顺势而为**、**做最肥的市场**、**"看五年想三年做好一两年"**

### 问：家里没钱，该不该为了理想去学艺术？

> 我说句不好听的，你别生气。
>
> 先谋生，再谋爱。你看我，给排水专业毕业的，我现在做手机、做汽车，跟给排水有半毛钱关系吗？没有。但给排水让我先活下来了。
>
> 选择比努力更重要，但"有得选"的前提是你足够努力，而且——你得先活着。

运用了**先谋生再谋爱**、**自身经历**、**克制贪婪**

更多对话示例在 [`leijun.skill/examples/demo-conversation.md`](leijun.skill/examples/demo-conversation.md)。

## 两个版本

| | leijun.skill | leijun.skill-rag |
|---|---|---|
| **定位** | 给别人用 | 你自己深度使用 |
| **SKILL.md** | 纯人设 | 人设 + 检索指令 |
| **知识来源** | 精炼在 SKILL.md | SKILL.md + 1038 个向量文档 |
| **依赖** | 无 | Python + ChromaDB + Embedding API |
| **触发方式** | `/leijun` | `/leijun-rag` |
| **检索** | 不支持 | 自动检索原始材料做事实支撑 |

两个版本独立完整，不互相依赖。leijun.skill 是 leijun.skill-rag 的子集。

## 蒸馏了什么

### 8个核心心智模型

| 模型 | 一句话 | 交叉验证 |
| --- | --- | --- |
| **顺势而为** | 选择比努力重要，站在风口上让势能推动你前进 | 6本书 + 15篇访谈 |
| **效率为王** | 小米的本质就是效率，砍掉中间环节，把实惠还给用户 | 4本书 + 8篇访谈 |
| **七字诀** | 专注、极致、口碑、快 | 7本书 + 全部访谈 |
| **用户信任** | 和用户交朋友，追求"闭着眼睛买"的终极信任 | 6本书 + 12篇访谈 |
| **克制贪婪** | 硬件净利润率不超过5%，用制度约束人性 | 5本书 + 8篇访谈 |
| **第一性原理** | 回到事物本质思考，不照搬已有经验 | 4本书 + 6篇访谈 |
| **爆品战略** | 少做产品，只做精品，满足80%用户的80%需求 | 4本书 + 5篇访谈 |
| **长期主义** | 机会比收入重要，成长比利润重要，伟大是熬出来的 | 4本书 + 10篇访谈 |

### 10条决策启发式

顺势而为 · 先想怎么输 · 十倍投入 · 守正出奇 · 对角线战略 · 超预期口碑 · 用时间换信任 · 效率优先 · 看五年想三年 · 成功大于盈亏

### 表达DNA

口语化 · 自黑精神 · 数据驱动 · 类比高手 · 故事→教训→原则→行动

### 5对内在张力

理想主义 vs 现实主义 · 激情 vs 克制 · 自信 vs 谦虚 · 长期 vs 短期 · 个人 vs 团队

## 构建过程

```
Phase 1：19 个 Agent 并行提取
  ├── 8 本书 Agent（每本 1 个，逐章提取）
  ├── 4 个访谈 Agent（22 篇分 4 批）
  ├── 3 个演讲 Agent（16 篇分 3 批）
  └── 3 个微信 Agent（16 篇分 3 批）
          ↓
Phase 2：4 个聚合 Agent 交叉验证
  ├── 书籍 → 心智模型（8 模型，多书交叉验证）
  ├── 访谈 → 决策启发式（12 条，附原文支撑）
  ├── 演讲+微信 → 价值观 + 表达DNA
  └── 全部 → 决策案例 + 时间线
          ↓
Phase 3：提炼 SKILL.md（269 行）
          ↓
Phase 4：质量验证 + 效果演示
```
